EMO
===========================

###########环境依赖
node-v8.9.4-x64.zip
redIs ~

###########部署步骤
1. npm install  //安装node运行环境

2. npm run dev   //前端编译

3. npm build 打包

4. 丢进支持html的容器就行列如 jboss ，tomcat ，nginx
