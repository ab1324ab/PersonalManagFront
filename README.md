**PersonalManagFront** EMO
===========================

########### `环境依赖`
node-v8.9.4-x64.zip\
redIs ~

########### `部署步骤`
1. _npm install_  `//安装node运行环境`

2. _npm run dev_  `//前端编译`

3. _npm build_ `打包`

4. _丢进支持html的容器就行列如 jboss ，tomcat ，nginx_
